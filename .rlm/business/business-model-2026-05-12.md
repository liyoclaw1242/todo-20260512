# Business Model Snapshot — 2026-05-12

**Project**: Personal todo web app with due-date visual alerts
**Signal**: Issue #3
**Date**: 2026-05-12

## Wedge

紙筆記事找不到、沒有視覺提醒。現有工具（Todoist 等）太重，想要一個開瀏覽器就能用的輕量 web app。

## Target User

liyo — 個人日常使用，self-use 工具，no monetisation。

## Status Quo

- 現在：紙筆 or 重量級 todo app（Todoist 等）
- 痛點：手寫找不到、重量級 app 介面複雜、缺乏截止日期視覺提醒
- 替換條件：開瀏覽器即用、截止日期一目了然、資料不消失

## Demand Reality

- 使用頻率：每日
- 關鍵行為：早上開瀏覽器→看到今天到期 + 過期項目→處理
- 不需要：協作、行動 app、複雜分類、通知推播

## Narrowest Wedge

最小版本：CRUD todo + 截止日期欄位 + 過期/今日標示 + 本地持久化。
搜尋是第一個擴充。

## Future Fit

個人工具，no monetisation，no growth pressure。保持輕量是長期目標。

## Business Model Classification

- Type: personal productivity tool (self-use)
- Revenue: none
- Scale: single user
- Monetisation: none
